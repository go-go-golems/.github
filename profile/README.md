### GO GO LLM AND AGENT RUNTIME

- [GO GO GOJA BLEVE](https://github.com/go-go-golems/goja-bleve) GO GO GOJA BLEVE IS A GO GO GADGET FOR FULL TEXT AND VECTOR SEARCH FROM JAVASCRIPT.
- [GO GO GOJA GIT](https://github.com/go-go-golems/goja-git) GO GO GOJA GIT IS A GO GO GADGET FOR DRIVING GIT FROM JAVASCRIPT.
- [GO GO GOJA GITHUB ACTIONS](https://github.com/go-go-golems/goja-github-actions) GO GO GOJA GHA IS A GO GO GADGET FOR RUNNING GITHUB ACTIONS STYLE JAVASCRIPT ON TOP OF GOJA.
- [GO GO GOJA TEXT](https://github.com/go-go-golems/goja-text) GO GO GOJA TEXT IS A GO GO GADGET FOR PARSING MARKDOWN, REPAIRING YAML AND JSON, AND EXTRACTING STRUCTURE FROM PROSE.
- [GO GO GOJA DBUS](https://github.com/go-go-golems/goja-dbus) GO GO GOJA DBUS IS A GO GO GADGET THAT IS STILL DREAMING OF BEING BORN (A PLANNED NATIVE MODULE FOR D-BUS).
